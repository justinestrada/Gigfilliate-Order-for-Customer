Gigfilliate WordPress plugin extension. This plugin requires the core Gigfilliate WordPress plugin. Gigfilliate allows any affiliate to place an order for a customer.

== Description ==

Affiliates can view, manage and order for their customers.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `gigfilliate-wp-customers.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
